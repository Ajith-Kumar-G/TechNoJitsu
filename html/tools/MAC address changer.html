MAC Address Changer

This is a novice level UNIX project that changes the MAC address of the specified network interface. This tool was developed by B V Vineeth.

To run this tool, you will need a Linux distro / Mac OS, or any UNIX Operating System as this tool calls certain system subprocesses specific to UNIX systems. Along with a UNIX system, you will also need python pakages to compile and execute commands.

You can find this project here: MAC address changer.

The MAC address changer is a simple tool that calls certain system subroutines to change the MAC address of the given network adapter. In order to use this tool, you need to know a few basic Linux shell commands. Here's what you will need to know to use this tool:

  • cd file_path / directory_path - Used to navigte to different directories
  • pwd - Returns the presently working directory
  • ls - Lists all files and subdirectories in the present directory
  • mkdir [option] dir_name - Creates a new directory
  • python3 file_name.py - Used to execute a python3/python file
  • python file_name.py - Used to execute a python file
Please note that the tool has support for both python and python3, so any of the 2 commands above can be used.

To use this tool, first navigate to the directory where the tool exists. Then run the script mac_changer.py in that directory. But before simply trying to run it, you must mention certain parameters to make the tool actually change the MAC address. These parameters are -i for interface, and -m for the new MAC address.

Two sample scenarios are shown below:

  • python3 mac_changer.py -i eth0 -m 00:11:22:33:44:55
  • python mac_changer.py -i wlan0 -m 0a:1b:2c:3d:4e:5f
Note that eth0 is the interface name in the first case, and wlan0 is the interface in the second case. The MAC addresses are mentioned after -m. They don't have to be the same as mentioned here, you can always modify it as you wish, but they must always be alphanumeric, and should be of 12 characters with every 2 characters seperated by a colon ':'. Use -help for more details on commands.
