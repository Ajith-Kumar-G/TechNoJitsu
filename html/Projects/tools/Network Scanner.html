Network Scanner

This is a novice level UNIX project that scans and returns all the clients present within the same IP range. This tool was developed by B V Vineeth.

To run this tool, you will need a Linux distro / Mac OS, or any UNIX Operating System as this tool calls certain system subprocesses specific to UNIX systems. Along with a UNIX system, you will also need python pakages to compile and execute commands.

You can find this project here: MAC address changer.

The Network Scanner is a simple tool that returns the MAC and IP addresses of all clients connected within the same target IP range. In order to use this tool, you need to know a few basic Linux shell commands. Here's what you will need to know to use this tool:

  • cd file_path / directory_path - Used to navigte to different directories
  • pwd - Returns the presently working directory
  • ls - Lists all files and subdirectories in the present directory
  • mkdir [option] dir_name - Creates a new directory
  • python3 file_name.py - Used to execute a python3/python file
  • python file_name.py - Used to execute a python file
Please note that the tool has support for both python and python3, so any of the 2 commands above can be used. Also, please make sure to use this on an updated version of Linux, with all python/python3 modules updated as well to avoid any unnecessary warnings/errors.

To use this tool, first navigate to the directory where the tool exists. Then run the script network_scanner.py in that directory. But before simply trying to run it, you need to mention the IP range within which the Scanner will scan with the parameter -t for target IP range.

Two sample scenarios are shown below:

  • python3 network_scanner.py -t 10.0.2.1/24
  • python3 network_scanner.py -t 10.0.2.1/16
Note that you need to specify the range as [IP]/24, /16, etc. The lower the range, the more packets will be transmitted.
