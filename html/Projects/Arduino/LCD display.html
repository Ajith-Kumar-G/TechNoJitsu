LCD Display

This is an intermediate level project to demonstrate the usage of a 16x2 LCD display with an Arduino. The project is meant to display the given text on the display. Please do not try this project straightaway if you don't have any prior experience with Arduino or don't know how to read schematic diagrams.

For this project, you will need :

  • 16x2 LCD display
  • Jumper cables
  • Arduino UNO R3
  • Breadboard
  • Potentiometer
  • Resistor
Please note that we have only recomended the above components based on reviews and pricing, and we do not expect you to stick to these components strictly.

To demonstrate this project our team has experimented using a 16x2 LCD display module (particularly JHD162A, though the variants have little or no difference in operations)

The potentiometer is used to adjust the contrast of the display. We have experimented using a 10k Ohm potentiometer, but 5k Ohm and 1k Ohm also work fine. Just adjust the contrast according to your convenience based on what you use by simply turning the knob.

In order to program this board after making the necessary connections, you will need the Arduino IDE. If this is your first time using the IDE, then you will need to set up your board on the IDE. Please follow this link to set up your board if it is your first time using an Arduino : How to set up Arduino board. Once you have set up your Arduino board, you must include the necessary libraries for that particular project. In this case, you need LedControl library for this project. Download the file and extract it to the same directory as your Arduino directory (By default C:\Users\usr\Documents\Arduino\libraries).

Now that we have everything set up, we can start programming our Arduino. You can find the code in our developer's GitHub repository: LCD Module Once entered, feel free to make any changes you wish while giving credits to the author for his contributions. After you made any / no changes to the code, save the file and keep in mind the file extension must be .imo as it is meant to be imported to your Arduino board. After compiling and uploading the file to your Arduino, you can notice a message shown on your LCD display. And this successfully completes this project.
